# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1wzwf8hDB2gnnymfkONz7dRx3UsRi_5es

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set `VITE_API_URL` in [.env.local](.env.local) (default: `http://127.0.0.1:8001`)
3. Run backend from `backend/` with `DATABASE_URL`, `JWT_SECRET`, and `GROQ_API_KEY` configured in `backend/.env`
4. Run the app:
   `npm run dev`
